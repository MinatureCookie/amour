addEventListener

addEventListener = function(name,
callback,
source)

Add a listener for some system event.  This should be used for events such as a mouse movement, or keyboard input

Parameters

nameThe name of the event to listen for
callbackA function to be triggered when the event is fired
sourceIf the event triggered specifies the same source, then this listener will be triggered.  Nil is for a global source

Send an event message.  This should be used for system events, such as mouse movement, or keyboard input

Parameters

nameThe name of the message we are sending
sourceOnly listeners specifying the same source will be triggered by this message.  Nil is for a global source
dataThe data payload to be used as a parameter to any activated listener’s callback
