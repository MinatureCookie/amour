addEventListener

addEventListener = function(name,
callback,
source)

Description

Adds a listener to the event queue.  The event queue should be used for system input, such as mouse events, keyboard events, etc.

Parameters

nameThe name of message we want to listen for
callbackThe method to call when we hear the message
sourceIf specified, will only activate if also specified in the message.  (Specifying a source means the message bus used is specific to the source, and is thus less congested)

addListener = function(name,
callback,
source)

Description

Adds a listener to the general queue.  The general queue should be used for events such as PLAYER_HAS_DIED, or FINISHED_DOWNLOADING_MUSIC

Parameters

nameThe name of message we want to listen for
callbackThe method to call when we hear the message
sourceIf specified, will only activate if also specified in the message.  (Specifying a source means the message bus used is specific to the source, and is thus less congested)
